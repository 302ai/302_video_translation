1. 🎥 多平台视频支持：观看YouTube、TikTok、Bilibili及抖音视频。
2. 🌍 多语言字幕翻译：轻松切换中文、英文、日文、德语、法语和韩语。
3. 📝 字幕格式下载：获取VTT、SRT、TXT格式的字幕文件。

## 技术栈
- Next.js 14
- Tailwind CSS
- Shadcn UI

## 开发&部署
1. 克隆项目 `git clone https://github.com/302ai/302_video_translation`
2. 安装依赖 `pnpm install`
3. 配置302的API KEY 参考.env.example
4. 运行项目 `pnpm dev`
5. 打包部署 `docker build -t video-translation . && docker run -p 3000:3000 video-translation`
